# Шебер — сервис поиска мастеров

«Шебер» — полнофункциональный сервис объявлений для поиска специалистов по ремонту и бытовым услугам в Казахстане.

## Возможности

- каталог мастеров с поиском, категориями, городами и сортировкой;
- регистрация и вход с безопасным хешированием паролей;
- личный кабинет, избранное и настройки профиля;
- создание, редактирование и удаление собственных объявлений;
- загрузка фотографий размером до 2 МБ;
- адаптивный интерфейс для компьютеров и мобильных устройств;
- единая SQLite-база пользователей, сессий, объявлений и избранного.

## Структура

```text
frontend/              статический интерфейс HTML, CSS и JavaScript
backend/server.js      Node.js API и раздача frontend
backend/db/schema.sql  документированная схема базы
backend/package.json   зависимости и команды сервера
.github/workflows/     публикация frontend на GitHub Pages
```

## Полнофункциональный запуск

Требуется Node.js 22.5 или новее.

```bash
cd backend
npm ci
npm start
```

Откройте `http://localhost:3000`. База `backend/stroy.db` создаётся автоматически и исключена из Git.

Переменная окружения `PORT` позволяет изменить порт сервера. Локальный `.env` хранится только на машине разработчика и не публикуется.

## GitHub Pages

Workflow `.github/workflows/pages.yml` автоматически публикует каталог `frontend/`. GitHub Pages показывает статический интерфейс и демонстрационные карточки, но не запускает Node.js API. Регистрация, общая база и изменение объявлений работают при открытии сайта через запущенный `backend/server.js` или после размещения backend на отдельном Node.js-хостинге.

## Безопасность

- пароли хешируются `crypto.scrypt` с индивидуальной солью;
- сессии используют криптографически случайные токены;
- SQL-запросы выполняются через подготовленные выражения;
- владельцы могут изменять и удалять только собственные объявления;
- базы, `.env`, исполняемые файлы, архивы и логи исключены из Git.
