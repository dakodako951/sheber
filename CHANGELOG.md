# История изменений

Все заметные изменения проекта фиксируются в этом файле. Версии оформляются по принципам семантического версионирования.

## [1.0.0] — 2026-07-10

### Добавлено

- адаптивная главная страница сервиса «Шебер»;
- каталог мастеров с поиском, категориями, фильтром по городу и сортировкой;
- пагинация результатов и режим просмотра избранного;
- демонстрационные регистрация, вход и выход из аккаунта;
- создание и удаление пользовательских объявлений;
- загрузка и предварительный просмотр фотографий;
- отдельная страница объявления с контактами и информацией о мастере;
- локальное хранение аккаунтов, сессии, объявлений и избранного;
- два русскоязычных файла README с обзором и подробной инструкцией.

### Примечание

Это первый публичный демонстрационный релиз. Серверная часть и база данных пока не используются.
