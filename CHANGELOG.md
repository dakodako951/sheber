# История изменений

Все заметные изменения проекта фиксируются в этом файле. Версии оформляются по принципам семантического версионирования.

## [2.0.0] — 2026-07-13

### Добавлено

- облачная регистрация и авторизация через Supabase Auth;
- общая база объявлений, доступная всем посетителям;
- личный кабинет пользователя с объявлениями, избранным и настройками;
- профильные страницы и адаптивные стили;
- SQL-схема таблиц, политик Row Level Security и хранилища изображений;
- операции создания, изменения и удаления объявлений через Supabase.

### Изменено

- локальные демонстрационные аккаунты заменены облачными;
- личный кабинет синхронизирован с текущей Supabase-сессией;
- локальная утилита `cloudflared.exe` исключена из репозитория.

### Безопасность

- доступ к профилям, объявлениям и избранному ограничен политиками RLS;
- в клиентском коде используется только публичный publishable key, без сервисных секретов.

## [1.1.0] — 2026-07-13

### Добавлено

- редактирование собственных объявлений непосредственно в каталоге;
- редактирование объявления на отдельной странице карточки;
- замена и удаление пользовательской фотографии;
- подтверждение закрытия формы при наличии несохранённых изменений;
- проверка владельца объявления перед сохранением изменений.

### Улучшено

- формы редактирования адаптированы под мобильные устройства;
- чтение пользовательских объявлений защищено от повреждённых данных в `localStorage`;
- архивы локальной сборки исключены из Git, чтобы не дублировать исходники.

## [1.0.1] — 2026-07-13

### Исправлено

- проект подготовлен к автоматической публикации на GitHub Pages;
- прямое открытие страницы объявления по параметру `?id=` больше не зависит от исходной вкладки;
- общие демонстрационные данные вынесены в отдельный статический файл;
- добавлена защита от повреждённых данных в локальном хранилище браузера;
- исправлены пути и кодировка мобильных подписей;
- добавлены метаданные темы, favicon и явные размеры логотипов.
- заменена недоступная фотография демонстрационного объявления;
- скрыто неактивное всплывающее уведомление, которое могло отображаться при полноэкранном захвате страницы;

### Оптимизировано

- размер двух локальных PNG-изображений уменьшен примерно на 78% без изменения внешнего вида;
- добавлены подсказки браузеру для приоритетной загрузки главного изображения.

## [1.0.0] — 2026-07-10

### Добавлено

- адаптивная главная страница сервиса «Шебер»;
- каталог мастеров с поиском, категориями, фильтром по городу и сортировкой;
- пагинация результатов и режим просмотра избранного;
- демонстрационные регистрация, вход и выход из аккаунта;
- создание и удаление пользовательских объявлений;
- загрузка и предварительный просмотр фотографий;
- отдельная страница объявления с контактами и информацией о мастере;
- локальное хранение аккаунтов, сессии, объявлений и избранного;
- два русскоязычных файла README с обзором и подробной инструкцией.

### Примечание

Это первый публичный демонстрационный релиз. Серверная часть и база данных пока не используются.
